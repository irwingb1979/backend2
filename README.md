## API Endpoints

The following API endpoints are available:

- **Hero Route**: 
  - `GET /api/hero` - Retrieve hero data.
  - `POST /api/hero` - Create a new hero entry.
  
- **Body Route**: 
  - `GET /api/body` - Retrieve body data.
  - `POST /api/body` - Create a new body entry.
